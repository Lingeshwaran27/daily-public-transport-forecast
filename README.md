# Daily Public Transport Forecast Project

This repo contains code, reports, and data for analysis and forecasting of daily public transport passenger journeys.

## Contents
- `notebooks/`: Full analysis & modeling Jupyter notebook
- `reports/`: Key findings, model comparison, insights, charts
- `data/`: Raw data needed for the forecast

## Quick Start
1. View full notebook in the `notebooks/` folder
2. Navigate to `reports/` for actionable insights and technical details
3. See `data/` for example inputs and outputs
